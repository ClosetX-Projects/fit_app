# **App Name**: FitAssist

## Core Features:

- User Registration and Profiles: Allow users to register, create profiles, and input personal data like name, age, gender, weight, height, and contact information (email, WhatsApp).
- Fitness Assessment Input: Enable users to record body composition measurements (weight, height, circumferences, skinfolds), bioimpedance data (age, height, gender, total body water, protein, mineral content, body fat mass), and strength test results (VO2max, 1RM test).
- Training Program Design: Create a module to design training programs based on pre-training assessments, recovery perception, and specific exercises tailored to the user.
- Workout Session Tracking: Allow users to log their workout sessions, including exercises performed, duration, volume (sets, reps, weight), Rate of Perceived Exertion (RPE), and intention to repeat.
- Progress Visualization: Provide charts and graphs to visualize user progress in body composition, strength, and workout consistency over time.
- AI-Powered Training Insights: The app will provide AI-generated insights based on the user's logged data, to make better training choices.
- Personalized Goal Recommendations: Suggest achievable and personalized fitness goals (e.g., weight loss, muscle gain) based on the user's assessment data and progress.  The LLM will act as a 'tool' for finding reasonable, motivating objectives.

## Style Guidelines:

- Primary color: Saturated cyan (#42A5F5) to evoke energy and a sense of progress.
- Background color: Light, desaturated cyan (#E1F5FE) for a clean and calming base.
- Accent color: Analogous blue-violet (#5C6BC0) to draw attention to key metrics and calls to action.
- Body and headline font: 'Inter' sans-serif for a clean, modern user interface.
- Use simple, geometric icons to represent different body measurements, exercises, and progress indicators.
- A clean and organized layout with a focus on data visualization. Use of cards to organize assessment inputs, and full-screen charts for tracking progress.
- Subtle transitions and animations on data updates.
