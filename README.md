# fit_app

This project was initialized from Firebase Studio.

Este projeto é uma plataforma completa para Personal Trainers e seus alunos, focada em monitoramento técnico e inteligência artificial.

## 🌟 Principais Funcionalidades Implementadas

### 🛡️ Gestão de Alunos e Convites
- **Convites Exclusivos**: Professores geram links que vinculam alunos automaticamente, impedindo cadastros sem supervisão.
- **Vínculo Direto**: O aluno já nasce dentro da base do professor, facilitando a prescrição imediata.

### 🧠 Inteligência Artificial (Google Gemini 2.5 Flash)
- **Mentor de Metas**: Planejamento personalizado baseado em bioimpedância, IMC e avaliações.
- **Coach IA**: Insights diários e dicas práticas baseadas no histórico recente.
- **Análise de Performance**: Insights técnicos sobre volume e intensidade de exercícios específicos.

### 📊 Monitoramento e Performance
- **Carga Interna**: Registro de PSE (Percepção Subjetiva de Esforço) com cálculo de carga (PSE x Tempo).
- **Lembrete de PSE**: Sistema que alerta o aluno para registrar o esforço após 15 minutos do treino.
- **Escala de Prazer (Feeling)**: Monitoramento do bem-estar psicológico pós-treino.

### 📋 Avaliação Física e Antropometria
- **Protocolo de 9 Dobras**: Registro detalhado para cálculo preciso de composição corporal.
- **Perímetros Bilaterais**: Acompanhamento de circunferências de membros superiores e inferiores.
- **Testes Funcionais**: VO2 Máx, TUG (Time Up and Go), Sentar e Levantar e Estimativa de 1RM.

### 🔔 Notificações e UX
- **Deep Linking**: Notificações que levam o aluno diretamente para a tela de preenchimento solicitada pelo professor.
- **Interface Mobile-First**: Navegação intuitiva via barra inferior para alunos e dashboard completo para professores.

## 🛠️ Stack Tecnológica
- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
- **UI**: Shadcn UI, Lucide Icons, Recharts.
- **Backend**: Firebase Firestore, Firebase Auth.
- **AI**: Genkit com Google Gemini 2.5 Flash.
